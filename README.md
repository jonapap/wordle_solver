# Wordle Solver

Simple program to quickly solve a wordle: https://www.powerlanguage.co.uk/wordle/.

The word list has been taken from http://www.mieliestronk.com/wordlist.html.

At each step, the word with the highest [entropy](https://en.wikipedia.org/wiki/Entropy_(information_theory)) is
selected to be the guess. See the video from 3Blue1Brown for more information:
https://www.youtube.com/watch?v=v68zYyaEmEA&t=723s
